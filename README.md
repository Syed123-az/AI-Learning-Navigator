# AI Learning Navigator

This project aims to build an AI-powered system that personalizes learning paths for students based on performance, engagement, and skill gaps.

## Features
- Student segmentation using clustering
- Performance prediction using ML
- Personalized learning recommendations
- Interactive dashboard

## Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- Streamlit / Power BI
